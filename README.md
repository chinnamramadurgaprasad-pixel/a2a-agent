# A2A-Enabled Agent Marketplace with MCP Gateway

A production-ready Agent-to-Agent (A2A) marketplace that enables AI agents to register, discover, and communicate with each other using standardized protocols. Includes MCP (Model Context Protocol) gateway support for integrating external tools.

## Features

- **Agent Registry Service**: Centralized directory where agents can register and be discovered
- **A2A Protocol**: Standardized JSON-based communication between agents
- **MCP Gateway**: Bridge between MCP tools and A2A agents
- **Multiple Agents**: Math helper and text summarizer agents with Azure OpenAI integration
- **Orchestrator**: Smart routing of tasks to appropriate agents
- **Fine-Tuning Support**: Infrastructure for customizing agent models
- **Streamlit UI**: User-friendly interface for browsing and testing agents
- **Docker Support**: Complete containerized deployment
